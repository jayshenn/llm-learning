# 第四阶段：机器学习 -- 智能时代的核心引擎

## 学习目标

理解机器学习核心算法原理，具备建模能力

## 核心内容

### 数学基础
- 微积分、线性代数、概率论

### 监督学习算法
- K 近邻（KNN）
- 线性回归（正规方程法、梯度下降）
- 逻辑回归（Sigmoid、损失函数、混淆矩阵）
- 决策树、朴素贝叶斯、支持向量机（SVM）
- 集成学习（AdaBoost）

### 无监督学习算法
- K-means 聚类

### 模型评估
- MAE、MSE、RMSE
- 精确率、召回率、F1-score
- ROC、AUC

### 特征工程
- 特征选择、特征降维（PCA）

## 目录结构

- `01-math/` - 数学基础

- `03-unsupervised/` - 无监督学习
- `04-feature-engineering/` - 特征工程
