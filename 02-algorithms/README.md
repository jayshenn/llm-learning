# 第二阶段：数据结构与算法 -- 人工智能的灵魂

## 学习目标

理解常见数据结构与算法，培养算法思维

## 核心内容

- 基础数据结构（链表、栈、队列）
- 排序算法（快排、归并、堆排序等）
- 查找算法（线性查找、二分查找、哈希查找）
- 高级算法思想（递归、动态规划、贪心、回溯）
- 时间复杂度与空间复杂度分析

## 目录结构

- `01-data-structures/` - 数据结构实现
- `02-sorting/` - 排序算法
- `03-problems/` - 算法题解

## 典型问题

- 数组问题、字符串问题、链表问题
- 查找问题、递归问题
- 动态规划问题、贪心问题、回溯问题
