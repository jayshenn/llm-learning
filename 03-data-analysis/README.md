## 核心内容

### Linux 基础
- 常用命令、Shell 脚本、环境配置

### MySQL
- SQL 语句、数据库设计、多表操作、开窗函数

- 数据导入导出、数据清洗、数据聚合、透视表

### 数据可视化
- Matplotlib、Seaborn、Pandas 绘图
