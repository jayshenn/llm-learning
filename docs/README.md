# 文档目录

本目录用于存放项目的正式文档，包括：

## 文档类型

### 学习笔记
- 各阶段的学习笔记
- 知识点总结
- 技术要点梳理

### 技术文档
- 算法原理说明
- 代码设计文档
- API 文档

### 项目文档
- 项目需求分析
- 系统设计文档
- 部署文档

## 文档规范

1. 所有文档使用 Markdown 格式
2. 文件命名采用小写字母，单词间用 `-` 分隔
3. 文档应包含清晰的标题和目录
4. 代码示例应使用代码块格式
5. 图表使用 Mermaid 或图片链接

## 目录结构建议

```
docs/
├── learning-notes/          # 学习笔记
│   ├── stage-01/
│   ├── stage-02/
│   └── ...
├── technical-docs/          # 技术文档
│   ├── algorithms/
│   ├── deep-learning/
│   └── ...
└── project-docs/            # 项目文档
    ├── project-01/
    ├── project-02/
    └── ...
```
