# 第五阶段：深度学习 -- 让 AI 像人类一样思考

## 学习目标

掌握深度学习框架和神经网络构建

## 核心内容

### PyTorch 基础
- 张量操作、自动微分、神经网络构建

### 神经网络
- 多层感知机（MLP）
- 激活函数（ReLU、Sigmoid、Tanh）
- 损失函数、反向传播算法
- 优化器（SGD、Adam）
- 正则化、Dropout

### 卷积神经网络（CNN）
- 卷积层、池化层、经典架构

### 循环神经网络（RNN）
- LSTM、GRU、词嵌入
